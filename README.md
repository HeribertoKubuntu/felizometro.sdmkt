# hery_nativefier

`hery_nativefier` es un proyecto para convertir aplicaciones web en apps de escritorio usando Electron, con enfoque en rendimiento para equipos de bajos recursos como Raspberry Pi.

## Para que sirve

- Ejecutar aplicaciones web como si fueran aplicaciones nativas.
- Reducir la carga de usar un navegador completo para cada servicio web.
- Unificar funcionalidades utiles como cache, reinicio, pantalla completa y actualizaciones.

## Fines del desarrollo

- Mejorar la experiencia de uso de apps web en entornos productivos.
- Facilitar empaquetado y distribucion para Linux, Windows y macOS.
- Permitir actualizaciones centralizadas usando GitHub Releases con `electron-updater`.
- Tener una base reusable para crear wrappers web ligeros y mantenibles.
